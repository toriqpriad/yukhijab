Cara Instalasi Bahasa Indonesia Opencart 2.0x

1. Extract "bahasa indonesia opencart 2.0x.rar"
2. Copy folder admin dan catalog ke direktori web Anda
3. Login ke Opencart Administration, klik System -> Localisation -> Languages,
4. Klik INSERT untuk membuat entry dalam untuk bahasa Indonesia. isikan parameter berikut:
   * Language Name : indonesia
   * Code : id
   * Locale : id_ID.UTF-8,id_ID,id-id,indonesia
   * Image : id.png
   * Directory : indonesia
   * Status : Enable
   * Sort Order : 2 (sort order bisa disesuaikan sendiri, bisa 0, 1, 2, dst)
4. Klik tab System -> Setting -> Edit (Toko Anda), pilih "Local". Kemudian pilih bahasa indonesia untuk Language * Klik "Save"
5. Katalog dan Admin Anda kini berbahasa indonesia.
